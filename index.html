📚 About the Project

The Boggle Game is a fast, exciting word-search experience built with Elixir. Players form words by connecting adjacent letters on a randomized 4x4 grid. This project showcases recursive search algorithms, dictionary validation, and functional programming paradigms.

✨ Features

Randomized Board

Every game is unique with a newly generated 4x4 letter grid.

Word Validation

Check player words against extensive real-world dictionaries.

DFS Search Logic

Uses depth-first search (DFS) and backtracking to find all valid paths.

Scalable Design

Optimized for speed, scalability, and ready for multiplayer extensions.

🎮 How to Play

  • Start the game by generating a randomized Boggle board.
  • Find words by connecting adjacent letters vertically, horizontally, or diagonally.
  • Submit your words and validate them against the dictionary.
  • Longer words score more points!
